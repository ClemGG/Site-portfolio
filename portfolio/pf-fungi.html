FUNGI

Rassasiez le vieux roi Fungi dans ce jeu d'arcane inspiré du cultissime Snake !

DESCRIPTION

Développement :

Date de sortie : Juin 2012
Moteur : Unity3D
Plateforme : Windows, Rasberry Pi
Lien : Github

Gameplay :

  Retombez en enfance dans ce jeu d'arcade inspiré du célèbre jeu Snake ; mais avec un twist ! La caméra se déplace de plus en plus vite d'un point à un autre de la carte. Utilisez vos pouvoirs pour ramasser le plus de champignons possible avant vos adversaires tout en restant dans le champ de la caméra. Si vous en sortez, c'est perdu ! Arriverez-vous à tenir le rythme ?
