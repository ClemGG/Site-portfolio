DIALOGUE NODE SYSTEM

Un système intuitif de création de dialogues par node pour Unity.

DESCRIPTION

Développement :

Date de sortie : Septembre 2021
Moteur : Unity3D
Lien : Github

  Le prototype d'un système de création de dialogue par node, réalisé avec Unity 2021.1.11.

  Contient :
 - Différentes nodes basiques avec la possibilité de créer des nodes personnalisées,
 - Support pour la création de fichiers de personnages et d'appel d'événements pour lier le dialogue a des effets visuels en jeu,
 - Support d'un système de sauvegarde et d'édition rapide de dialogue via un fichier CSV,
 - Une scène de test avec une interface préconfigurée pouvant utiliser des entrées clavier et manette,
 - Et bien plus de choses à retrouver sur mon Github !
