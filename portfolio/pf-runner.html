PAINT RUNNER

Un Endless Runner sur le thème "Passage" réalisé pour l'Extra Credits Game Jam n°5.

DESCRIPTION

Développement :

Date de sortie : Août 2019
Moteur : Unity3D
Plateforme : Windows
Lien : Github, Itch.io

Gameplay :

  Dans cet endless runner, mélangez les couleurs pour modifier votre vaisseau afin de briser les obstacles. Réalisé en deux jours dans le but de tester un script de recyclage de terrain infini.

(Ce jeu est accessible aux daltoniens)
