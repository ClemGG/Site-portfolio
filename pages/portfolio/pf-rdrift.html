RDrift

Courez contre la montre (mais aussi la rage !)
dans ce jeu de plateforme réalisé pour la Brackeys Game Jam n°3.

DESCRIPTION

Développement :

Date de sortie : Février 2020
Moteur : Unity3D
Plateforme : Windows
Lien : Github, Itch.io

Gameplay :

  Collectez le plus de pièces possible dans le temps imparti dans une multitude de niveaux mettant au test votre équilibre (et surtout votre patience !) dans une course contre la montre extrêmement difficile.
