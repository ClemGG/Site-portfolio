LA COMMUNE

Combattez les Maraudeurs volant les ressources de la Commune dans ce jeu de défense
se déroulant dans une Russie post-ère glaciaire.

DESCRIPTION

Développement :

Date de sortie : Juin 2018
Moteur : Unity3D
Plateforme : Windows
Lien : Github

Synopsis :

  Combattez les Maraudeurs aux côtés de la Commune dans une Russie fictive dévastée par une nouvelle ère glaciaire !

  Seules quelques poches de l'humanité ont pu survivre grâce au Magma, une nouvelle forme d'énergie quasi-illimité capable d'alimenter toute une colonie depuis les profondeurs de la Terre. Mais cette source d'énergie attire des convoitises...


Gameplay :

  Construisez et améliorez vos tourelles pour abattre vos ennemis ! Chaque nouveau type d'ennemi arrive par vague au milieu d'événements aléatoires (blizzards, montées de Magma), vous forçant à toujours adapter vos constructions pour tirer le meilleur parti de vos ressources limitées !
